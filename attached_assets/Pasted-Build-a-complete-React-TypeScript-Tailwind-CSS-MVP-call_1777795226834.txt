Build a complete React + TypeScript + Tailwind CSS MVP called NeedMates.

NeedMates is a local social trading and connection platform where people post what they need locally, find others who need the same thing, get help from trusted neighbours, and invite local businesses to respond.

It is not a normal marketplace. It is a local demand network.

Core positioning:
“Post what you need. Find people nearby who can help, join in, or recommend someone trusted.”

Important context:
This project is being built first in Replit, but the final codebase will later be moved into Lovable. Build the code cleanly so Lovable can continue from it. Use simple React/TypeScript patterns, clear component boundaries, reusable components, centralised mock data, and standard Tailwind classes. Avoid clever abstractions, obscure libraries, over-engineering, or deeply nested type inference.

Tech stack:
- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- lucide-react
- Local mock data in TypeScript files

Visual direction:
- Clean, warm, Notion-like interface
- Soft off-white or pale background
- Rounded white cards
- Subtle shadows
- Strong spacing
- Clear typography
- Friendly icons
- Polished mobile responsiveness
- Deep green, teal, warm blue, or similar as primary colour
- Accent colours used sparingly
- No generic Bootstrap look
- No harsh colours
- No clutter
- No cheap classified ads feel

Quality bar:
This should look and feel like a real product demo, not a rough prototype.
The homepage should explain the idea in under 10 seconds.
The feed should feel active.
The “I need this too” mechanic should be obvious.
Group demand should feel like the main differentiator.
Trust signals should be visible across the product.
The code should be clean enough for Lovable to continue later.
The app must be responsive.
